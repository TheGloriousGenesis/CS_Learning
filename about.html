About

This site is a personal knowledge base for learning computer science and software development. It's powered by GitHub Pages and dynamically loads blog posts directly from Markdown files located within a GitHub repository.

Features

  • Markdown rendering using markdown-it with support for admonitions like [!NOTE]
  • Dark/light theme toggle
  • Clean card layout for blog previews
  • Pagination, search by tag, and rich content styling
